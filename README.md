# ECSE546: Advanced Image Synthesis

This is the main repository containing the files for the course ECSE546: Advanced Image Synthesis course at McGill University. 

The repository mainly contains the C++ code for the concepts of image synthesis, Monte-Carlo Algorithm, and ray tracing theories, along with some course slides, notes, and written assignments. Copyright of course slides belong to Prof. Derek Nowrouzezahrai at McGill University. 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Microsoft Visual C++ 2017 x64. The programs are mainly developed with Visual Studio 2017.
